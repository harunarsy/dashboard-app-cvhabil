# 🚀 V0.1 RELEASE - SIMPLE STEP-BY-STEP

**Tujuan:** Create official V0.1 release dengan changelog  
**Waktu:** ~15-20 menit  
**Difficulty:** Easy ✅

---

## STEP 1: Copy CHANGELOG.md ke Project

### Option A: Manual (Gampang)
1. Buka file **CHANGELOG.md** dari outputs saya
2. Copy semua isinya
3. Di project folder kamu, buat file baru: `CHANGELOG.md`
4. Paste isinya
5. Save

### Option B: Via Terminal
```bash
cd ~/Users/harunarasyid/Projects/dashboard-app

# Paste file dari outputs ke sini
# (Copy CHANGELOG.md dari outputs ke folder ini)
```

**Verify:** File CHANGELOG.md sudah ada di root folder project kamu ✅

---

## STEP 2: Commit CHANGELOG ke Git

```bash
cd ~/Users/harunarasyid/Projects/dashboard-app

# Add file
git add CHANGELOG.md

# Commit
git commit -m "docs: add CHANGELOG.md for v0.1.0 release"

# Verify
git log --oneline
```

Harusnya muncul commit baru di atas dengan message "docs: add CHANGELOG.md"

---

## STEP 3: Create Git Tag

```bash
# Create tag
git tag -a v0.1.0 -m "Dashboard CV Habil V0.1 - MVP Release"

# Verify tag created
git tag -l
```

Harusnya muncul `v0.1.0` dalam list

---

## STEP 4: Push ke GitHub

```bash
# Push commits
git push origin main

# Push tags
git push origin v0.1.0
```

Tunggu sampai selesai...

---

## STEP 5: Create Release di GitHub

1. Buka: https://github.com/harunarsy/dashboard-app-cvhabil
2. Klik tab **"Releases"**
3. Klik tombol **"Create a new release"** (berwarna biru)

---

## STEP 6: Fill Release Form

**Choose a tag:**
- Pilih dropdown → pilih `v0.1.0`

**Release title:**
```
Dashboard CV Habil V0.1 - MVP Release
```

**Describe this release:**
Copy-paste dari CHANGELOG.md isinya (dari baris `## [0.1.0]` sampai bagian **Next Steps**).

---

## STEP 7: Publish

1. Biarkan **"Set as the latest release"** ✅ (jika ada checkbox)
2. Klik tombol **"Publish release"** (hijau)

---

## ✅ DONE!

Selesai! Kamu sudah punya V0.1 release 🎉

## ❓ Kalau Error?

### Error: "fatal: a tag 'v0.1.0' already exists"
```bash
# Delete existing tag
git tag -d v0.1.0
# Try again
git tag -a v0.1.0 -m "Dashboard CV Habil V0.1 - MVP Release"
```

### Error: "everything up-to-date"
Artinya sudah push, itu bagus! Lanjut ke STEP 5

### Error: "remote: Permission denied"
Pastikan sudah login GitHub dengan akun yang benar
